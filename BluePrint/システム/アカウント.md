#アカウント

## サインイン

+ 必要なパラメータ
  + メールアドレス
  + パスワード
+ 出力パラメータ
  + トークン

## サインアップ

+ 入力
  + ユーザ名
  + メールアドレス
  + パスワード
+ 出力
  + トークン
+ 保存
  + データベースへの登録

## ユーザー情報取得

+ 入力
  + トークン
+ 出力
  + ユーザ名

他のクラスから呼び出せるクラスとして実装。

実行するとトークンがある場合はユーザ名を返す。
